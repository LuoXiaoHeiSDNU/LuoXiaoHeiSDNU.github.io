密钥协商

密钥协商协议是用于两个用户一起确定一个共同使用的密钥，常见的密钥协商协议由Diffie-Hellman密钥协商协议，其可以简述如下：

  1. 密钥生成：$( G , g , q ) \rightarrow ( S K _ { n } ,g ^ { S K _ { n } } )$，每个用户使用密钥生成协议生成自己的密钥$S K _ { n }$，并将$g ^ { S K _ { n } }$发送给另一个参与方$m$。
  2. 密钥协商： $( S K _ { n } , g ^ { S K m } ) \rightarrow S _ { n , m }$，参与方$m$使用$g ^ { S K _ { n } }$与自己的密钥$S K _ { m }$生成公共密钥$S _ { n , m }$

通常在实际的应用中我们设置公共密钥$S _ { n , m }$为 $H((g^{SK_{m}})^{SK_{n}})$

秘密分享

N-t秘密分享协议可以分为两部分：

  1. 秘密分享阶段：$( s , t , \mathcal{U} ) \rightarrow \{ ( n , s _ { n } ) \} _ { n \in \mathcal{U} }$表示将秘密值$s$分为$n$份，每个用户拿到的只是秘密的份额部分，即$s _ { n }$，任何小于$t$的用户都无法恢复出原始的秘密值$s$。
  2. 秘密恢复阶段：$( \{ ( n , s _ { n } ) \} _ { n \in M } , t ) \rightarrow s $ 表示任何多于$t$的用户都可以在一起恢复出原始的秘密值$s$。

伪随机函数·伪随机生成器

伪随机函数是为了解决在计算机中没有真正的随机数生成器而提出的，常见的伪随机函数有很多，这里可以据一些例子：

双密钥伪随机函数：

首先我们先选择一个密钥对 $K = ({K_1},{K_2})$作为伪随机函数的密钥，我们的为随机函数则可以写为： $P{F_K}:{\{ 0,1\} ^} \times {\{ 0,1\} ^} \to {G_1}\times{G_2}$

该函数由两部分构成分别为：$PF_{K_{1}}: \left\{ 0,1 \right\} ^{}\rightarrow {G_1}$以及 $PF_{K_{2}}: \left\{ 0,1 \right\} ^{}\rightarrow {G_2}$构成

我们输入 $({I_1},{I_2})$到 $P{F_K}$中，可以得到如下的结果：

$PF_{K_{1}}(I_{1})=(\gamma _{I_1},v_{I_{1}})$ 以及 $ PF_{K_{2}}(I_{2})=(\gamma _{I{2}},v_{I_{2}})$，进一步而言我们可以得到

$$ P{F_K}({I_1},{I_2}) = (E,F) = ({g^{{\gamma {{I_1}}}{\gamma {{I_2}}} + {v{{I_1}}}{v{{I_2}}}}},{h^{{\gamma {{I_1}}}{\gamma {{I_2}}} + {v{{I_1}}}{v{{I_2}}}}}) $$

其中的$g,h$分别为群${G_1},{G_2}$的生成元。

伪随机生成器要求，伪随机生成器在种子的帮助下生成的概率分布应该和随机数的概率分布在空间上不可区分的。

双线性配对

(8 封私信 / 6 条消息) 对称双线性配对有如下性质吗？ - 知乎 (zhihu.com)

首先简单的介绍一下双线性配对的定义，其可以概括为，将两个群上的元素通过某种运算映射到一个新的群中。

双线性配对可以分为两种：两个乘法群运算到一个乘法群中去以及两个加法循环群映射到一个乘法循环群中去。

  • 如果是两个加法循环群映射到一个乘法循环群，双线性对可以按下列方式描述：

    设${G_1}$、${G_2}$为两个$q$阶的加法循环群，其中$q$是大素数，若映射：$e:{G_1} \times {G_2} \to {G_T}$（${G_T}$是一个$q$阶的乘法循环群）满足以下的性质，则$e$是一个双线性对：

    • 双线性：对于任意的$P \in {G_1},Q \in {G_2}$，任意的$a,b \in {Z_q}$，有$e(aP,bQ) = e{(P,Q)^{ab}}$，其中${Z_q}$是$q$阶整数循环群
    • 非退化性：若$P$为${G_1}$ 的生成元，$Q$为${G_2}$的生成元，则$e(P,Q)$为${G_T}$的生成元
    • 可计算性：对于任意的$P \in {G_1},Q \in {G_2}$，有高效的算法可以计算$e(P,Q)$。

  • 如果是两个乘法循环群映射到一个乘法循环群，双线性对可以按下面的方式描述： 设 ${G_1},{G_2}$为两个$q$阶的加法循环群，其中$q$为大素数。若映射 $e:{G_1} \times {G_2} \to {G_T}$（${G_T}$为$q$阶的乘法循环群）满足以下的性质，则$e$是一个双线性对：

    • 双线性：对于任意的 ${g_1} \in {G_1},{g_2} \in {G_2}$，任意 $a,b \in {Z_q}$，有 $e(g_1^a,g_2^b) = e{({g_1},{g_2})^{ab}}$其中 ${Z_q}$为$q$阶循环群
    • 非退化性：若 ${g_1}$为 ${G_1}$生成元， ${g_2}$为 ${G_2}$生成元，则$e(g_1^a,g_2^b)$为 ${G_T}$生成元。
    • 可计算性：对于任意的 ${g_1} \in {G_1},{g_2} \in {G_2}$，有高效的算法可以计算$e(g_1^a,g_2^b)$。
